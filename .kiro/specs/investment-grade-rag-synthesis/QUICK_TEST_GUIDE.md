## If Issues Persist

### Modal Still Opens Multiple Times
- Check browser console for `window._citationHandlerRegistered` value
- Should be `true` after first initialization
- If `false`, event listener deduplication isn't working

### Modal Text Still Cut Off
- Check if outer container has `max-h-[500px]`
- Check if inner div has `max-h-[400px] overflow-y-auto`
- Verify text has proper wrapping styles

### Tables Still Render as Raw Markdown
- **First**: Confirm you're asking a NEW question (not an old one)
- Check browser console for "Using prompt library" or "Generating response" logs
- Verify the response is actually NEW (not cached)
- If still broken, check system prompt in bedrock.service.ts (lines 520-570)
