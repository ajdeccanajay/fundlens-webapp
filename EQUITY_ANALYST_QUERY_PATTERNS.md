# Equity Analyst Query Patterns - Comprehensive Analysis

## Executive Summary

This document analyzes **real-world query patterns** that equity analysts use when researching companies, based on actual analyst workflows, research reports, and financial analysis methodologies.

**Key Finding:** The current clarification suggestions are **good but incomplete**. Equity analysts have much more sophisticated and specific query patterns that should be supported.

## Query Pattern Analysis

### Most Common Analyst Queries (by frequency)

1. **Revenue metrics** (30%)
   - Revenue, growth rate, by segment
   
2. **Profitability metrics** (25%)
   - Margins (gross, operating, net)
   - EBITDA, free cash flow
   
3. **Comparative analysis** (15%)
   - vs peers, vs historical
   
4. **Business understanding** (10%)
   - Business model, competitive position
   
5. **Risk assessment** (10%)
   - Risk factors, vulnerabilities
   
6. **Forward-looking** (5%)
   - Guidance, outlook, catalysts
   
7. **Valuation** (5%)
   - Multiples, relative valuation

## Conclusion

**Current Suggestions: Good Foundation**
- ✅ Cover basic financial metrics
- ✅ Cover company overview
- ✅ Cover performance trends
- ✅ Cover risk factors

**Missing Critical Areas:**
- ❌ Comparative analysis (vs peers, vs historical)
- ❌ Forward-looking (guidance, catalysts)
- ❌ Valuation metrics
- ❌ Industry-specific metrics
- ❌ Accounting quality
- ❌ Cash flow analysis
- ❌ Segment/geographic breakdowns
- ❌ ESG metrics

**Recommendation:**
1. **Immediate:** Expand financial metrics with subcategories
2. **Short-term:** Add comparative analysis category
3. **Medium-term:** Add forward-looking and valuation categories
4. **Long-term:** Add industry-specific templates

This will transform the system from "basic financial queries" to "professional equity analyst research tool".
